This section will help you get started with `flake-file` in minutes.

It pretty much depends on your current Nix choices -the library adapts to you-
and what your destination is.

## Choose your path

<Card title="Sandboxed Bootstrap" icon="nix">
  Try flake-file in a temporary directory using your flake inputs.

  <LinkButton variant="secondary" icon="right-arrow" href="/tutorials/bootstrap">Try It Now!</LinkButton>

<Card title="Brand new project (with or without flakes)" icon="rocket">

  <LinkButton variant="secondary" href="/guides/templates" icon="right-arrow">Pick one</LinkButton> that suits you more, and run:

  ```shell
  nix flake init -t github:vic/flake-file#default
  ```

</Card>
